maven环境变量配置

Posted at 2019-07-13 笔记 

第一步 下载

从官网下载自己想要的maven版本，然后解压。
官网

第二步 配环境变量

1、添加系统环境变量
name: M2_HOME
value:指向解压的地址
2、添加系统环境变量
name: MAVEN_HOME
value:指向解压的地址
3、添加path路径
value: %M2_HOME%/bin
注:经过path之后,可以在cmd中运行bin文件夹中的软件。
测试:cmd + mvn -v
输出:版本，jdk,地址等信息便可
问题: M2_HOME和MAVEN_HOME两个之间有什么不同？？

第三步 更改映像地址

定位到解压的目录,进入conf文件夹，打开setting.xml文件
在<mirrors>标签中添加新的映像地址
例如:阿里的地址

至此，安装和配置环境变量完成

Maven使用命令

1、启动spring boot
mvn spring-boot:run

2、创建项目命令
mvn archetype:generate

3、将项目转成idea项目
mvn idea:idea

4、编译项目
mvn complie

5、清除旧的class文件
mvn clean

6、生成测试报告
mvn test

7、打包
mvn package

8、安装到仓库
mvn install
