hexo 基本用法

Posted at 2019-03-11

常用的hexo命令

hexo new “newpostname” 新建文章

缩写：hexo n
hexo new page “newpagename”

hexo generate 生成静态界面至public目录

缩写: hexo g

hexo server 打开预览端口

缩写：hexo s

hexo deploy 部署到github中

缩写：hexo d

新建笔记配置

在指定目录生成对应tag的文章

npm 使用淘宝映像

npm install --registry=https://registry.npm.taobao.org
