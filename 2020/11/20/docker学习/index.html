Posted at 2020-11-20 docker 

docker 常用命令

映像相关操作

  • 获得在线映像$docker pull imagename:verision

容器相关操作

  • 创建容器:$docker run -itd --name haha -p 9999:8080 centos:6.7 /bin/bash, -i:交互式操作,-t:终端,-d:后台运行,指定--name容器名称为:haha,-p 8080:8080将容器中的8080端口映射为宿主机的9999端口.
  • 查看当前所有容器的状态:$docker ps -a
  • 启动容器:$docker start [id/names]
  • 停止容器:$docker stop [id/names]
  • 重启容器:$docker restart [id/names]
  • 进入容器:$docker attach [id/names]或$docker exec -it [id/names] /bin/bash,第一种方式退出容器的时候容器也会退出.
  • 查看容器日志:$docker logs -f [id/names]
  • 删除容器:$docker rm -f [id/names]
